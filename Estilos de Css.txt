Estilos de Css

Inline: É o estilo mais básico de HTML nele você faz as edições dentro das próprias tags como: <p style: "color"= blue> lorem </p>, é o menos usado porém é o mais pontual, use apenas se quiser modficar uma tag específica.

Interno: É um estilo melhor porém só afeta uma página, perfeito quando se quer modificar uma página específica, é um estilo melhor e deve ser iniciado com uma tag <style> dento da área <head>, podendo ser abaixo ou acima da tag <title>

EX: <head> <title> aaaaaa </title>
 <style>
        body {
            background-color: cornflowerblue; font-size: 20px; font-family: Arial, Helvetica, sans-serif; text-align: justify;
        }
        h1 {
            color: brown; text-align: center;
        }
        h2 {
            color: blueviolet;text-align: left;
        }
        p {
            color: white; text-align: justify;
        }
    </style>
</head>

Externo: O melhor estilo, afeta todas as páginas, para ser utilizado basta criar uma tag link: css dentro da área <head> e criar uma página para esta área, podendo utilizar do atalho ctrl + clique que abre uma nova página ao clicar em cima do nome da página que se quer criar, antes de fazer as modificações necessárias é bom usar da regra do UTF-8 usando a tag @charset "UTF-8";
